# 日報作成アプリ

## 概要
業務の日報を入力・管理するための簡易アプリです。  
出勤日、作業内容、成長点、反省点、月次総評を記録できます。

GitHub上で成果物として確認できるように作成しました。

## 使用技術
- Java  
- （必要なら）Google Apps Script  
- Google スプレッドシート（データ管理）

## 使い方
1. 入力画面を開く
2. 日付・作業内容などを入力
3. 送信すると内容が保存されます

※ ローカル環境での実行は想定していません。

## ファイル構成
daily-report-app/
├─ README.md # 本ファイル
├─ index.html # 入力画面

└─ service/ # 業務ロジック

- 週単位・月単位で振り返りができる構成にした
